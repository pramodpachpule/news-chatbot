# News Chatbot - Client

A React-based frontend for a RAG-powered news chatbot.

## Features

- Real-time chat interface
- Session management
- Responsive design
- Error handling

## Prerequisites

- Node.js (v18 or higher)
- npm or yarn
- Backend server running (see server README)
