# News Chatbot - Backend

A FastAPI backend support for a RAG-powered news chatbot.

## Features
- Session management
- RAG pipeline integration
- Redis caching
- Jina embeddings

## Prerequisites

- Python 3.10+
- Redis server
- Jina API key
- Google Gemini API key
